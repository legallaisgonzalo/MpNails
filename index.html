Tocá en los servicios para saber de que se trata

  • Esmaltado semipermanente

    • Esmaltado semipermanente es un gel acrílico mezclado con esmalte
    que se aplica sobre la uña y se endurece con rayos ultravioleta. Tiene la misma textura que un esmalte clásico: es líquido y tiene incluso más brillo que el tradicional. Se mantiene impecable, sin deslustrarse ni descamarse

  • Kapping gel

    • Se trata de un sistema innovador que está indicado para aquellas personas con uñas naturales que no desean o no puedan tener un full set de uñas esculpidas; es un excelente tratamiento para uñas débiles, quebradizas o escamadas

  • Uñas esculpidas

    • Las uñas esculpidas son aquellas que se hacen a partir de materiales como acrílico y gel que permiten moldear la forma y longitud de cada uña. Este tipo de manicura se hace desde cero, es decir que a través de moldes y productos específicos se moldea la uña, dándole la forma y longitud más adecuada.
